# WeatherApp
## Tutorial Design + Code

Es un app que muesttra el estado del tiempo que se esta presentando en una determinada ubicación. El proyecto utiliza los siguientes tecnologías:

- SwiftUI
- Core Location
- AsyncImage (iOS 15+)
- LocationButton (iOS 15+)
